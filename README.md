m2p2-repository
===============

garc33's maven repository

How to use it
=============

Just add to your settings.xml
```xml
<repositories>
	<repository>
		<id>garc33-releases</id>
		<url>https://raw.github.com/garc33/m2p2-repository/mvn-repo/maven/releases/</url>
	</repository>
</repositories>
```
